Research CSS3 Animations

What are the general recommendations for animations?

Animations are used to change an element gradually from one style into another. Keyframes, which hold what styles the element will have at given times, must first be specified for the animation. As many CSS properties can be changed as desired and however many times as desired.

What issues need to be considered and addressed when using animations?

Keyframes must be specified to use animation, and vendor prefixes are need to run animations in certain internet browsers.
