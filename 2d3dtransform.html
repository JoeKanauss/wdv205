Research 2d and 3d Transformations

1. What are the general recommendations for 2D transformations?

2d transformations are used to change element shapes, sizes, and positions with rotate, skew, scale, and translate.

2. What issues need to be considered and addressed when using 2D transformations?

2d transformations require a certain vendor prefix depending on which internet browser is being used. These vendor prefixes are included in the html comments for this page.

3. What are the general recommendations for 3D transformations?

3d transformations are used to rotate the x, y, and/or z coordinates of an element.

4. What issues need to be considered and address when using 3D transformations?

3d transformations require a certain vendor prefix depending on which internet browser is being used. These vendor prefixes are included in the html comments for this page.
