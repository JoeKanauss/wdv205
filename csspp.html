Define CSS Preprocessor

What does a CSS Preprocessor do?

A CSS Preprocessor allows CSS code to be written in a more efficient manner.

What are the advantages of using a CSS Preprocessor?

Reduced redundancy, smart styles, reuse of code among many projects.

What are the disadvantages or issues you need to consider when using a CSS Preprocessor?

More steps, loss of ability to change live pages quickly.

List at least FIVE CSS Preprocessors currently available.

SASS, LESS, Stylus, CSS Crush, Myth.
